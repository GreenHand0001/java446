# 教务信息管理系统 java446

本项目是一款基于 Vue3 和 Nodejs 的教务信息管理系统，采用前后端分离的设计模式。系统致力于为教育机构提供一个高效、易用的管理平台。

## 技术栈

- **前端：**
  - Vue3
  - Element-plus

- **后端：**
  - Nodejs 14
  - Express

- **数据库：**
  - MySQL 5.7

## 功能模块

系统根据不同身份，提供不同的功能模块。

### 学生模块

- 登陆注册
- 查看个人成绩
- 修改个人信息
- 查看公告

### 教师模块

- 查看学生信息
- 录入成绩
- 成绩分析
- 打印成绩
- 申请开放
- 表格上传成绩
- 查看公告
- 修改个人信息

### 管理员模块

- 教师管理
- 学生管理
- 成绩管理
- 公告管理
- 设置教师录入成绩时间
- 修改密码
- 查看申请

## 系统角色

系统定义了以下三种角色：

- 学生
- 教师
- 管理员

各角色拥有不同的操作权限和功能模块。

## 运行环境

- 前端：现代浏览器
- 后端：Nodejs 14 或以上版本
- 数据库：MySQL 5.7 或以上版本

## 部署步骤

1. 安装数据库（MySQL 5.7）
2. 导入数据库结构
3. 部署后端服务（Nodejs 14+）
4. 部署前端应用（Vue3）

## 目录结构

```
.
├── frontend/           # 前端代码
│   ├── public/         # 公共资源
│   ├── src/            # 源码
│   ├── package.json    # 前端项目配置文件
│   └── ...
└── backend/            # 后端代码

    ├── routes/         # 路由
    ├── controllers/    # 控制器
    ├── package.json    # 后端项目配置文件
    └── ...
```

## 核心亮点

- 前后端分离，便于维护和扩展
- 基于角色的权限管理，确保系统安全性和数据隔离
- 成绩录入与分析功能，助力教育管理决策
- 高度模块化设计，易于二次开发与定制
